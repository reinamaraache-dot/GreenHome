# GreenHome — Milestone 1

A smart plant-care web application to help users manage and track indoor/outdoor plants.

## Technologies
- HTML, CSS, JS
- PHP (later phases)
- MySQL database
- GitHub for version control

## Milestone 1 Includes
- Basic pages (Home, Login, Register, Dashboard, Plants)
- Responsive design (white + green modern theme)
- Initial database schema
- Project folder structure
